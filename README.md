# homebrain

HomeBrain is an early-stage project for a family-oriented long-term memory
brain.

The project is intended to help a household keep durable, searchable memory
over time. The implementation is still being prepared, and the first public API
will be documented here when it is ready.

## Install

```sh
npm install homebrain
```

## Status

This package currently publishes the project namespace and basic package
metadata. Functional releases will follow as the implementation takes shape.
