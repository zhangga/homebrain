# homeagent

HomeAgent is an early-stage package for home-oriented AI agent capabilities.

It is intended to become part of the HomeBrain project, helping a household use
durable memory through practical agents and workflows. The implementation is
still being prepared, and the first public API will be documented here when it
is ready.

## Install

```sh
npm install homeagent
```

## Status

This package currently publishes the project namespace and basic package
metadata. Functional releases will follow as the implementation takes shape.
